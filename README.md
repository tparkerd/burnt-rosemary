# burnt-rosemary
Data importation modules for Baxter lab's GWAS database

## Installation

### Environment
Make a `.env` file in your root directory. You may use the `.env.example` as a basis for it.

```ini
# .env.example
database=baxdb
user=baxdb_owner
password=password
host=localhost
port=5432
```

## Usage
    python import.p
